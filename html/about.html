More about the API

This section is dedicated to informing you all about the API that was used to make this website

What is the Fixer API?

The Fixer API is an easy to use API for current and historical data values

What you can do?

You can get the latest currency rates and historical current rates. Their API endpoints allow you to convert from one currency to another, use symbols to incorporate in your data and so much more!
